# R Package for Analysis Reproducibility of lncRNA-Ebola Project
## Projects
- [lncRNA](https://github.com/FernandoMarcon/Ebola_lncRNA)

#### Usage:
```R
devtools::install_github('FernandoMarcon/REbola')
library(REbola)

#--- Load Preprocessed Datasets
# Geneva
data(geneva)

# USA
data(usa)

#--- Load Raw Datasets
# Geneva
data(raw_geneva)

# USA
data(raw_usa)

```
